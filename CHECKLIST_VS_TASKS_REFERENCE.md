# Checklist vs Tasks - Quick Reference

## Simple Checklist (NEW - Improved)
`
 Buy groceries
 Call dentist
 Finish report (strikethrough when checked)
 Send email
`

**Features:**
- Simple checkbox + text
- Strikethrough when checked
- No extra metadata
- Fast and lightweight
- Toggle on/off like lists

**Use Cases:**
- Shopping lists
- Quick to-do items
- Meeting notes
- Brainstorming lists
- Simple action items

---

## Tasks (Existing - Complex)
`
 Launch new product                 [HIGH] [ Dec 31] [ Monthly] [ Reminder]
   Complete documentation          [MEDIUM]
   Review final designs            [LOW]
 Quarterly review meeting          [HIGH] [ Completed]
`

**Features:**
- Checkbox + text + metadata
- Due dates
- Priority levels (Low/Medium/High)
- Recurring patterns
- Reminders
- Nested tasks
- Visual badges and indicators

**Use Cases:**
- Project management
- Complex workflows
- Team collaboration
- Deadline tracking
- Recurring obligations

---

## When to Use What?

**Use Simple Checklist when:**
- Taking quick notes
- Making shopping/packing lists
- Brainstorming ideas
- Simple action items
- You don't need dates or priorities

**Use Tasks when:**
- Managing projects
- Tracking deadlines
- Need priorities
- Want recurring tasks
- Team collaboration
- Complex workflows

---

## Technical Differences

| Feature | Simple Checklist | Tasks |
|---------|-----------------|-------|
| Checkbox Type | HTML native | Custom styled |
| Metadata | None | Extensive |
| UI Complexity | Minimal | Feature-rich |
| Performance | Fast | Moderate |
| Toggle Mode | Yes | No |
| Strikethrough | Yes | Yes |
| Badges/Tags | No | Yes |
| Nested Items | No | Yes |
